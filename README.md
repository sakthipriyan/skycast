#skycast


##Environment tested
### Java 1.7.0_40
### Apache Maven 3.0.4

##Cloning the code
```
git clone https://github.com/spriyan/skycast.git
```

##Installing mvn repo
```
mvn clean install
```

##Running the example
```
mvn exec:java -Dexec.mainClass=com.sakthipriyan.skycast.SkycastApp -Dexec.args="'1 100' '4 78 79 80 3' '8 10 13 13 100 99 98 77 81'"
```
